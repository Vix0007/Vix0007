### What I'm building

**AEGIS** — Adversarial Entropy-Guided Immune System.

A physics-based network intrusion detection system that detects adversarial traffic through flow thermodynamics — zero payload inspection. No byte reading. No DPI. Just the mathematics of how automated systems move differently from humans.

- **F1: 0.9952 · TPR: 99.50% · FPR: 0.21% · Inference: 262μs**
- Architecture: TVD-HL-SSM (Thermodynamic Variance-Guided Hyperbolic Liquid State Space Model) with Mamba-3 core
- Published: [arXiv:2604.02149](https://arxiv.org/abs/2604.02149) · Dataset: [Vix0007/AEGIS-Adversarial-Corpus](https://huggingface.co/datasets/Vix0007/AEGIS-Adversarial-Corpus) (908K sequences, 10GB)
- Organically indexed in English, German, and Chinese within 8 days of publication
